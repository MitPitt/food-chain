# Практикум на ЭВМ, задача М2.2 (Э). Трофическая цепь.

Нужные библиотеки:

- `pip install pyqt5`
- `pip install matplotlib==3.2.2`
- `pip install scipy`

Для сборки в исполняемый файл с помощью Pyinstaller:

- `pip install pyinstaller`
- `pyinstaller .\Application.spec`
